Detailed Description

Declarations and definitions related to the fontCharacter_t structure and its operations.

This header file provides the declarations for the fontCharacter_t structure and its associated functions. The fontCharacter_t represents a font character defined by a sequence of strokes. Each stroke indicates a movement of the pen (with pen up or pen down states) to draw the character. The file also declares the fontCharConstructor function which creates and initializes a fontCharacter_t instance.

typedef struct fontCharacter_s fontCharacter_t

Structure representing a font character.

This structure stores all the strokes that form a single character. It keeps track of:

  • The ASCII character it represents.
  • The total number of strokes.
  • The current stroke index (to which new strokes can be appended).
  • A dynamically allocated array of strokes.

It also provides function pointers to:

  • Append a new stroke to the character.
  • Free all associated resources when the character is no longer needed.

◆ stroke_t

typedef struct stroke_s stroke_t

Structure representing a single stroke of a character.

A stroke consists of a vector (defining direction and length) and a pen state. The pen state indicates whether the pen is down (drawing) or up (not drawing) as it moves along the specified vector.

A pointer to the newly created fontCharacter_t object, or NULL if allocation fails.

This function allocates memory for a fontCharacter_t structure, initializes its fields, and allocates memory for the specified number of strokes. If memory allocation fails, an error is reported, and NULL is returned. The caller must eventually call the free function pointer within the structure to properly deallocate the memory.
