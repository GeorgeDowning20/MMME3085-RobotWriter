Detailed Description

Declarations for robot control functions and related constants.

This file provides the interfaces for controlling the robot's position and movements. It includes functions to start up the robot, send strokes (pen movements) to it, and return it to a 'home' position, as well as defining various constants that govern its coordinate space and text dimensions.

Moves the robot to its home position.

Returns
SUCCESS on success, or an appropriate error code if moving to home fails.

Moves the robot to a defined home position by sending the appropriate G-code command. After constructing the G-code command with the home coordinates, it sends the command to the robot and prints the command for debugging purposes.

Sends a stroke command to the robot based on the current cursor position.

Parameters
[in]cursorPointer to the cursor_t structure representing the robot's current position.
[in]strokeThe stroke_t structure containing the vector and pen state to apply.
Returns
SUCCESS on success, or an appropriate error code if sending the stroke fails.

Sends a stroke command to the robot based on the provided cursor position and stroke data. The stroke includes a vector (defining the movement direction and distance) and a pen state (up or down). Depending on the pen state, the command uses either a rapid move (G0) with no drawing or a linear move (G1) with drawing. After forming the command, the function sends it to the robot and prints the command for debugging.

◆ StartUpRobot()

errorCode_t StartUpRobot ( void  )

Initializes and starts up the robot.

Returns
SUCCESS on successful startup, or an appropriate error code if startup fails.

Initializes the robot by attempting to open the RS232 port and then sending a sequence of commands to prepare the robot. It sets the robot to a known position, starts the spindle or pen movement, sets initial speed parameters, and finally moves the robot to the home position. If the COM port cannot be opened, it reports an error.
